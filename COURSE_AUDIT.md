# Course Audit

Last reviewed: 2026-05-22

## Fixed in this pass

- Simulator coverage was uneven. Air Brakes, Combination, Cargo, and the endorsement simulators now have enough available questions for their advertised run lengths.
- Practice mode now pulls from the broader simulator bank, so topic practice is deeper than the lesson quick checks.
- Simulator results had a state-order bug that could break the final score screen. The finished simulator type and label are now preserved before the attempt is cleared.
- The simulator panel wording now describes multiple exam simulators instead of one 100-question simulator.
- The Sources tab no longer links to the local handbook extraction file on GitHub Pages, where that private source file is not published.
